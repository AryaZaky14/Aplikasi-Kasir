# Aplikasi Kasir

## Deskripsi
Aplikasi kasir berbasis web yang dikembangkan menggunakan Laravel sebagai implementasi mata kuliah Pemrograman Web. Sistem digunakan untuk mengelola transaksi, produk, kategori, dan pengguna.

## Fitur
- Login
- Dashboard Admin
- CRUD Produk
- CRUD Kategori
- CRUD Transaksi
- Laporan

## Teknologi
- Laravel
- PHP
- MySQL
